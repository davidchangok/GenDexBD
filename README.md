# GenDexBD

WoW 12.0 (Midnight) 宠物对战品种识别与最优属性追踪插件。

## 功能

### 1. 品种识别

自动推算宠物品种（P/S、P/P、H/H 等），基于属性比例匹配，兼容各品质等级。

### 2. 最优属性管理

**纯手动标记**每种宠物的最佳品种，支持按场景分类（PvP/PvE/收藏/自定义）并附加自由备注。最优属性的定义权完全属于用户，插件绝不自动判定。

**右键菜单**（整合进 Rematch 宠物列表右键菜单）：

- 一级菜单列出该物种**所有已设最佳品种**（金黄色 ★ 标记），当前宠物匹配的品种可直接取消，非匹配品种可点击删除
- 尚未设置任何最佳品种时，灰色提示"尚未设置最佳品种"
- 当前宠物的品种可一键设为最佳
- 唯一属性物种（仅一种品种变体）直接提示"唯一属性即为最佳"
- "设为其他属性"子菜单**仅显示当前物种实际存在的品种**（通过 Rematch `possibleBreedIDs` 获取）

### 3. 战斗提示

- 野外宠物对战中，遇到已标记为目标品种的宠物时，屏幕中央显示金色醒目提示
- 目标品种的敌方宠物框体上显示金色 ★ 标记（品种级精确匹配）
- 可以在设置中调整提示显示时长（1–30 秒）

### 4. 遇敌统计

记录野外对战中遇到的每种宠物的每个品种的遇敌次数，在设置面板中以表格形式展示（按宠物→品种汇总排序）。

### 5. 设置面板（/gbbd）

双标签页设计：

- **常规设置**：鼠标提示、战斗提示、遇敌计数等开关 + 导出/导入 + 遇敌统计表格
- **最优品种**：已保存的最优品种列表（宠物名称、品种、宠物类型），按物种名称排序

### 6. 导入导出

支持序列化导出/导入最优品种数据，方便备份和分享。导出格式为 `speciesID=breedID|category|note`。

## 命令

- `/gbbd` — 打开设置面板

## 设计原则

GenDexBD **绝不自动判定**哪种品种是"最优"。每种宠物、每个场景下"最优"的含义都不同，最优属性的定义权完全属于用户。所有数据优先从 Rematch 获取，不重复造轮子。
