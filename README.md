# GenDexBD

WoW 12.0 (Midnight) 宠物对战品种识别与最优属性追踪插件。

## 功能

1. **品种识别** — 自动推算宠物的 Breed（S/S、P/P、H/H 等）
2. **最优属性管理** — 纯手动标记每种宠物的最佳品种，支持按场景分类（PvP/PvE/收藏/自定义）并附加自由备注
3. **战斗提示** — 遇到目标品种时屏幕中央金色醒目提醒

## 使用

- `/genedex` 或 `/gd` — 打开配置面板

## 设计原则

GenDexBD **绝不自动判定**哪种品种是"最优"。每种宠物、每个场景下"最优"的含义都不同，最优属性的定义权完全属于用户。

## 兼容性

- 目标版本: WoW 12.0 (Midnight), Interface 120007
- 理论兼容: 11.x (Dragonflight), 10.x (The War Within)
