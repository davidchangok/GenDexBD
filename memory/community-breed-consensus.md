## 家族级社区共识（用于未索引宠物参考）

见 [[project-state]] 算法参数部分的家族被动修正。以下家族有明确品种倾向：

- **兔子/野兔/松鼠/花栗鼠/老鼠(Flurry系)/狐狸**: S/S
- **赞达拉/钢铁星弹**: P/P
- **绵羊/羔羊/蜗牛**: H/P
- **螃蟹/龟**: H/H(PvE坦克) 或 P/P(PvP输出)
- **青蛙/蟾蜍**: H/P(治疗) 或 S/S(控制),分两派
- **蜘蛛**: S/S(控场) 或 P/P(爆发),因品种而定
- **蝙蝠/猫/熊崽**: P/P
- **亡灵幽灵**: 因宠而异
- **蛾类**: P/P或P/S（飞行被动给速度）
- **蟑螂**: S/S
- **土拨鼠**: P/B（自加速→S/S冗余）
- **软泥**: P/P
- **机械**: P/P(通用)
- **15个冷门家族**: 无社区讨论（鹿/豪猪/水獭/企鹅/水黾/鹤/鱼/乌贼/树苗/猪/黄貂鱼/蜻蜓/双足飞龙/石头/山羊）
