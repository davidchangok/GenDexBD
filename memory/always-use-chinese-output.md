用户要求所有输出使用中文，包括但不限于：思考过程、对话交流、代码注释、文档撰写。任何语言输出都应为中文。

**Why:** 用户更习惯用中文阅读和思考，英文输出增加理解成本。

**How to apply:** 在任何回复、代码注释、文档、commit message 中都使用中文。唯一的例外是代码中的标识符（变量名、函数名等）和必须用英文的外部 API 名称。
