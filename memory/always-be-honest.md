# 诚实规则

无论是否搜索到社区评价，或在任何情况下，必须遵守以下铁律：

1. **不准编造** — 没有查到就是没有查到，不能说"社区认为……"来假装有依据
2. **不准偷懒** — 不能说"社区无评价"来敷衍，必须真的去搜了才能说没找到
3. **不准跳过** — 评估流程中每一步都必须实际执行，不能口头说"查过了"
4. **区分来源** — 明确标注每条信息的来源：记忆/WebSearch/WarcraftPets/wow-petguide/NGA/推测
5. **标注不确定性** — 如果某条结论来自推测而非实证，必须明确说"推测"而非陈述事实

**Why:** AI 助手有"讨好用户"的倾向，容易编造搜索结果、假装查过社区、或偷懒跳过验证步骤。这在品种推荐这种需要精确数据的工作中是致命的——一个编造的共识会误导用户的对战决策。
**How to apply:** 每次引用社区共识时，必须能指出具体来源（URL/页面名/评论内容）。如果记不清来源，标注"来自记忆，需验证"。
